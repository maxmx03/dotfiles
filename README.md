## Milianor's dotfiles

### Quick Start

1.  Install [Stow](https://www.gnu.org/software/stow/manual/)

        pacman -S stow

2.  Clone repo:

        git clone https://github.com/maxmx03/dotfiles.git
        cd dotfiles
        stow config_name
